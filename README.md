# American Sign Language Recognition for Emergency Alerts: A Machine Learning Approach

## 📖 Description
This project is a real-time sign language recognition system that converts hand gestures into text and audio output. It is designed to assist communication for deaf and dumb individuals, especially in emergency situations.

## 🚀 Features
- Real-time gesture recognition
- Converts gestures into text
- Audio output generation
- Machine learning-based classification

## 🛠 Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- VGG16

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the project:
   python main.py
